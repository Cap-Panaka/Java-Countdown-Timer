# ⏳ Java Countdown Timer

A clean, minimalist, and efficient console-based countdown timer built with Java. This tool allows users to set a duration in seconds and provides real-time feedback until completion.

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)
![Status](https://img.shields.io/badge/Status-Complete-green?style=for-the-badge)

## 🌟 Features

- **Real-time Feedback**: Precise 1-second interval updates using `java.util.TimerTask`.
- **Interactive Input**: Easy-to-use CLI interface for setting countdown durations.
- **Visual Completion**: Displays a celebratory "Happy New Year" message when the timer hits zero.
- **Resource Management**: Automatically cancels the timer and releases system resources upon completion.

## 🚀 Getting Started

### Prerequisites

- [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/) 8 or higher.

### Installation & Execution

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Cap-Panaka/Java-Countdown-Timer.git
   cd Java-Countdown-Timer
   ```

2. **Compile the program**:
   ```bash
   javac Main.java
   ```

3. **Run the countdown**:
   ```bash
   java Main
   ```

## 🛠️ Built With

- **Java**: The core programming language.
- **Timer & TimerTask**: Utilized for efficient scheduling and background task execution at fixed rates.
- **Scanner**: Leveraged for robust user input handling.

## 📂 Project Structure

- `Main.java`: Contains the primary logic for input handling, timer initialization, and the background countdown loop.
